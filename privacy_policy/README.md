This folder contains the privacy policy page for the **Smart Storage Cleaner** Android app.

## 📁 Files
| File | Description |
|------|-------------|
| `index.html` | The privacy policy web page (dark themed, mobile-friendly) |

---

## 🚀 How to Host on GitHub Pages

Follow these steps to get a live URL for your Play Store listing:

### Step 1 — Create a GitHub Repository
1. Go to [github.com](https://github.com) and sign in
2. Click **+ New repository**
3. Name it: `smart-storage-cleaner-privacy`
4. Set it to **Public**
5. Click **Create repository**

### Step 2 — Upload the File
1. In the new repo, click **Add file → Upload files**
2. Upload `index.html` from this folder
3. Click **Commit changes**

### Step 3 — Enable GitHub Pages
1. Go to the repo **Settings** tab
2. Scroll to **Pages** (left sidebar)
3. Under **Source**, select `main` branch and `/ (root)` folder
4. Click **Save**

### Step 4 — Get Your URL
After ~1 minute, your privacy policy will be live at:
```
https://YOUR_GITHUB_USERNAME.github.io/smart-storage-cleaner-privacy/
```

### Step 5 — Update the Contact Email
Before publishing, open `index.html` and replace:
```
your@email.com
```
With your actual email address.

---

## 📱 Use in Play Store
Paste your GitHub Pages URL in:
**Google Play Console → App Content → Privacy Policy**

## 📲 Add to App (Optional)
You can add a "Privacy Policy" button in your Flutter app's settings screen that opens this URL using the `url_launcher` package.
